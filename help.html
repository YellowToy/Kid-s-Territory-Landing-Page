HTML Structure

This landing page is a flexible layout with three columns built using framework Bootstrap 3.x. All of the information within the main content area is nested within a div with an class of "contaner-fluid". All the next code divided into sections using div with id names "about-us", "methods", "gallery", "teachers", "prices". The general template structure is the same throughout the template. Here is the general structure.

If you would like to edit the color, font, or style of any elements in one of these columns, you would do the following:

 #primaryContent a {
                  color: #someColor;
                }

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight.

I.E.

 #wrap #primaryContent a {
                  color: #someColor;
                }

So, to ensure that your new styles are applied, make sure that they carry enough "weight" and that there isn't a style lower in the CSS file that is being applied after yours.

CSS Structure

I'm using two CSS files in this landing page. The first one is a bootstrap file. This file also contains some general styling, such as anchor tag colors, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the file.

The second file contains all of the specific stylings for the page. The file is separated into sections using:

 /* === Header Section === */

                some code

                /* === Slider === */
                
                some code
                
                /* === About === */
                
                some code
                
                /* === Methods === */
                
                some code
                
                etc, etc.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

For instance, if you would like to edit the display of these images, find the following section in the style sheet:

 #gallery img {
                  change styles here:
                }

  2. Most of the animation in this site is carried out with the help of css rules. But I use some custom scrips for several animations. There are a few functions written in Javascript. 
            $('.carousel').carousel({

PHP Code Explanation

From php code I use only contact.php for running the subscription form. Although the file includes all the code with comments, I will explain some main points you would like to adjust.

            /* Set e-mail recipient */
            $myemail  = "you@domain.com";

            /* Let's prepare the message for the e-mail */
            $message = "Hello!

            Your contact form has been submitted by:
            Name: $name
            E-mail: $email

            End of message
            ";

For changing the recipient email you change the line $myemail = "you@domain.com" and change the value for desired email.

If you want to change the text of a message, you go to the variable $message and put your text as you want them to look like. Pay attention to the "" signs. All your message should go between them.

Assets

In the website folder a number of other forders: css, fonts, img, js and video.

'Fonts' and 'Video' Folders

Folders 'fonts' and 'video' shouldn't be a problem. They contain only those fonts and video I use in the site.

If you like to change fonts I suggest you go to style.css and change the direcotry for them at the very beginning of a document. I prefer Google fonts.

            @import url(http://fonts.googleapis.com/css?family=Pacifico);
            @import url(http://fonts.googleapis.com/css?family=Lora);

For changing video you go to index.html and change directory (src) there too. Pay attention to the type of your video. Not all browsers suppor all types of video.

          source src="video/Envato_Studio.mp4" type="video/mp4"
